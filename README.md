# YouTube video search on Vue.js

Light-weight web page for simplified search of videos on youtube. Built especially for the vehicles where you can open the webpage and watch the video on fullscreen.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
